# Boiler Plate Express DDD

I create this boilerplate for my self. Not perfect, but good for everyone who wants to start learning about Domain Driven Design.

## Environment

```
    - Node V14
    - PostgreSQL
    - Redis
    - Express
    - Dotenv
    - etc.
```

## Installation

```
npm install
```

## Usage

```
npm start
```
